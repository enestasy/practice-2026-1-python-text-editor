🤝 Проект-партнёр: Издательство «Экивоки»

Проект выполнен в сотрудничестве с издательством настольных игр «Экивоки». Редактор DnDEditor учитывает потребности мастеров подземелий и может быть интегрирован в образовательные программы издательства.

  • 📄 Отчёт о взаимодействии с партнёром (Markdown) Описание формата сотрудничества, полученных знаний и связи с проектом «Подземелья Политеха».

📚 Техническая документация

  • 📖 README.md — Описание проекта Краткое введение, инструкция по запуску, структура репозитория.
  • 🛠 TECHNICAL_GUIDE.md — Техническое руководство Пошаговая инструкция по созданию редактора, архитектура, UML-диаграммы, описание модификаций.
  • 👥 Индивидуальные планы участников Детальное распределение задач между Труфановой А.М. и Руссковой В.А.
