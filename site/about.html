📖 Описание проекта

D&D Scenario Editor — специализированный текстовый редактор для мастеров подземелий, разработанный в рамках проектной практики Московского Политеха (группа 251-333). Приложение предназначено для создания, редактирования и структурирования сценариев настольных ролевых игр Dungeons & Dragons.

Проект выполнен в связке с учебным проектом «Подземелья Политеха» и учитывает потребности партнёра — издательства настольных игр «Экивоки». Редактор помогает мастерам быстро создавать локации, NPC, боевые сцены и диалоги с визуальной подсветкой ключевых элементов.

🎯 Цели и задачи

  • Разработать удобный инструмент для ведения заметок во время D&D-сессий
  • Реализовать подсветку синтаксиса для D&D-тегов ([БОЙ], [ЛУТ], [NPC] и др.)
  • Создать библиотеку шаблонов для быстрого старта сцен
  • Добавить встроенный генератор бросков кубиков (d4–d100)
  • Обеспечить экспорт сценариев в HTML для использования на стримах
  • Освоить командную работу через Git/GitHub

⚙️ Технологический стек

Python 3.10+ tkinter (GUI) Git/GitHub Markdown HTML/CSS

✨ Ключевые возможности

🎨 Подсветка тегов

8 категорий D&D-элементов подсвечиваются разными цветами: бои (красный), проверки (оранжевый), лут (зелёный), NPC (фиолетовый) и др.

📝 Шаблоны сцен

5 готовых шаблонов: локация, NPC, боевая сцена, сокровище, диалог. Вставляются в один клик через меню.

🎲 Генератор кубиков

Встроенный рандомайзер для d4, d6, d8, d10, d12, d20, d100. Результат автоматически вставляется в текст.

📊 Статистика сценария

Модальное окно показывает количество локаций, NPC, боёв, проверок и других элементов в текущем файле.

💾 Экспорт в HTML

Конверсия Markdown-разметки в веб-страницу с сохранением цветовой схемы тегов. Идеально для стримов.

🌙 Тёмная тема

Переключение между светлой и тёмной палитрами интерфейса по Ctrl+T. Комфорт для глаз при ночных сессиях.

🏗 Архитектура приложения

Приложение построено на основе ООП-подхода. Основной класс DnDEditor инкапсулирует логику GUI, обработки событий и управления состоянием. Подсветка синтаксиса реализована через динамическое применение тегов tkinter с использованием регулярных выражений.
