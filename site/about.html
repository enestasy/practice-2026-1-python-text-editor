Описание проекта

Цель проекта — изучить основы GUI-разработки на Python, работу с файловой системой и организацию кода в рамках командной практики. Редактор создавался с нуля на основе tutorial, затем был расширен собственными модификациями.

Технологический стек

  • Python 3.12
  • tkinter (встроенная GUI-библиотека)
  • Git/GitHub для контроля версий
  • Markdown для документации
