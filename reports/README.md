# Отчёты в бинарных форматах

В этой папке размещены итоговые отчёты по проектной практике в бинарных форматах.

## Файлы отчётов

| Файл | Описание |
|------|----------|
| [report.docx](report.docx) | Итоговый отчёт по проектной практике в формате Microsoft Word (DOCX). Содержит титульный лист, введение, описание выполненной работы, результаты взаимодействия с организацией-партнёром, выводы и приложения. |
| [report.pdf](report.pdf) | PDF-версия итогового отчёта. Полностью идентичен DOCX-файлу, предназначен для загрузки в СДО (LMS) и печати. |

## Шаблон отчёта

Структура и оформление отчёта выполнены в соответствии с шаблоном, предоставленным кафедрой:

- [practice_report_template.docx](../task/report_template.docx) — шаблон отчёта (размещён в папке `task/`)

## Дополнительные документы

Помимо бинарных отчётов, в репозитории также присутствуют детализированные материалы в формате Markdown:

- [individual_plans.md](individual_plans.md) — индивидуальные планы работы участников группы
- [partner_report.md](partner_report.md) — подробный отчёт о взаимодействии с организацией-партнёром «Подземелья Политеха»
