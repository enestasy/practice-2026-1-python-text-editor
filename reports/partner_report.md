## 🔹 Характер взаимодействия

В рамках проектной практики прямые очные или онлайн-встречи с представителями организации-партнёра не проводились в связи с форматом организации учебного процесса. Взаимодействие осуществлялось через:

1. **Официальные учебные ресурсы:**
   - Шаблон репозитория `practice-2026-1` на GitHub
   - Задания и критерии оценки, размещённые в СДО (LMS)
   - Методические материалы по дисциплине «Проектная деятельность»

2. **Кураторскую поддержку:**
   - Информация о проекте «Подземелья Политеха» от куратора М. Баркар
   - Консультации по структуре отчёта и требованиям к документации

3. **Самостоятельное изучение:**
   - Анализ требований к проекту на основе предоставленных документов
   - Исследование сторонних ресурсов (Instructables, документация Python) для реализации технического задания

## 🔹 Связь с проектом «Подземелья Политеха»

Хотя текстовый редактор является самостоятельным учебным продуктом, его разработка способствовала достижению целей основного проекта:

1. **Практическая ценность:** Созданный редактор может использоваться как:
   - Учебный пример для новых участников проекта
   - Инструмент для ведения заметок и документации внутри команды
   - Демонстрация подхода «build your own X» для образовательных целей

2. **Методическая ценность:** 
   - Отработана структура репозитория, которую можно тиражировать
   - Создан шаблон технической документации (`TECHNICAL_GUIDE.md`)
   - Зафиксированы типовые ошибки и способы их решения

3. **Компетенции команды:** 
   - Участники приобрели навыки, необходимые для разработки функциональных модулей в рамках «Подземелий Политеха»
   - Освоили цикл: исследование → реализация → документирование → презентация
