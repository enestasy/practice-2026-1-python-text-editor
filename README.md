## Вариативная часть задания

**Тематика:** Python: Create a Simple Python Text Editor!  
**Источник:** [Instructables Tutorial](http://www.instructables.com/id/Create-a-Simple-Python-Text-Editor/)  
**Описание:** Разработка простого текстового редактора на языке Python с использованием встроенной библиотеки tkinter. В рамках задания проведено исследование предметной области, воспроизведена базовая функциональность (открытие, сохранение, редактирование файлов), а также реализована творческая модификация проекта (добавление строки состояния с подсчётом слов и символов, настройка горячих клавиш). Результаты исследования и инструкция по созданию оформлены в виде технического руководства.
