## Вариативная часть задания

**Тематика:** Python: Create a Simple Python Text Editor!  
**Источник:** [Instructables Tutorial](http://www.instructables.com/id/Create-a-Simple-Python-Text-Editor/)

**Описание:** Разработка специализированного GUI-редактора сценариев для настольных ролевых игр (D&D) на языке Python с использованием стандартной библиотеки `tkinter`. В рамках проектной практики проведено исследование требований мастеров подземелий, воспроизведена базовая функциональность текстового процессора (работа с файлами, кодировка UTF-8), а также реализована уникальная модификация проекта: внедрена система динамической подсветки синтаксиса для 8 категорий игровых тегов, поддержка встроенной Markdown-разметки, библиотека шаблонов сцен, генератор бросков кубиков и экспорт в HTML с сохранением цветовой структуры. Результаты исследования, архитектура приложения и пошаговая инструкция по разработке оформлены в виде подробного технического руководства.
